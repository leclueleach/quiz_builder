Convert a Word or PowerPoint quiz

To convert a Word or PowerPoint quiz file into this builder:

  1. Copy the prompt below.
  2. Open Claude or ChatGPT.
  3. Paste the prompt and upload your .docx or .pptx file.
  4. Download the HTML file the AI returns.
  5. Use Import Standalone HTML above to load it into the builder.
